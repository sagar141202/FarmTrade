# CropChain — Farmer DeFi Intelligence Layer

CropChain is an AI-powered agricultural intelligence platform designed to help farmers predict crop yields, detect unfair market prices, and connect with investors.

## Core Features

Crop Yield Predictor
- Linear Regression model using weather + soil data

Fair Price Radar
- Isolation Forest anomaly detection for mandi price exploitation

Negotiation Coach
- Groq Llama 3 LLM generates farmer negotiation scripts

Investor Marketplace
- Farmers publish ML verified crop investment proposals

Mobile App
- Built using Capacitor and distributed as Android APK

## Tech Stack

Frontend
React + Vite + Tailwind + Zustand + Recharts + Framer Motion

Backend
FastAPI + PostgreSQL + SQLAlchemy + Alembic

AI
scikit-learn ML models + Groq LLM

Infrastructure
Supabase + Railway + Vercel + GitHub Actions
